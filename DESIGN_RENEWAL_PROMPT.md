# Book Quest — 디자인 컬러 리뉴얼 요청

## 요청 사항

현재 앱의 색상이 너무 알록달록해서 독서 앱이라는 느낌이 부족해.
차분하고 어두운 초록색 기반으로 전체 색감을 통일해줘.
"조용한 서재에서 책 읽는 느낌"을 주고 싶어.

## 디자인 방향

- **핵심 키워드:** 차분함, 고급스러움, 클래식한 서재, 깊은 초록
- **참고 이미지 느낌:** 오래된 가죽 책, 어두운 원목 서재, 초록 독서등, 양피지
- RPG 판타지 느낌은 유지하되, 색감만 차분하게 정돈

## 적용 범위

### 전체
- 헤더 배경, 탭 바, 카드, 버튼, 프로그레스 바 등 모든 UI 요소에 적용
- 기존의 파란색(#5B8DEF) 포인트 → 포레스트 그린(#3D5A3E)으로 교체
- 기존의 밝은 녹색(#66BB6A) → 자연 초록(#5B8C5A)으로 교체
- 기존의 강한 노란색 골드(#FFB300) → 앤틱 골드(#C4933F)로 교체

### 장르 태그
- 기존의 원색 배경 태그 → 채도 낮은 색상으로 변경
- 태그 배경은 해당 색상의 10% 투명도, 텍스트는 해당 색상 100%

### 퀘스트 패널
- 일일/주간/월간 탭 색상도 초록 계열로 통일
  - 일일: #5B8C5A (자연 초록)
  - 주간: #4A7A8A (차분한 청록)
  - 월간: #8B6F47 (가죽 갈색)

### 캐릭터 화면
- 캐릭터 배경 그라데이션: 아이보리 → 연한 초록 (라이트) / 짙은 초록 → 검정 (다크)
- 스탯 레이더 차트: 채도 낮은 장르 색상 적용

### 상점
- 장비 등급 색상은 기존 유지 (Iron~Challenger 등급 구분이 중요하므로)
- 단, 주변 UI(버튼, 배경, 테두리)는 새 팔레트 적용

### 업적/통계
- 달성 업적 배경: 연한 초록 (기존 연한 녹색 유지하되 채도 낮춤)
- 차트 바/그래프: 새 팔레트의 포인트 색상 적용

## 폰트 느낌 (선택사항)

만약 폰트도 바꿀 수 있다면, 클래식한 세리프 폰트를 타이틀에 적용하면 서재 느낌이 더 살아.
- 타이틀/헤더: Noto Serif KR 또는 KoPub Batang
- 본문/UI: 기존 시스템 폰트 유지
