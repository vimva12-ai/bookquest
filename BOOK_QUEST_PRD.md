## 1. 프로젝트 개요

### 앱 설명
"책을 읽을수록 캐릭터가 성장하는 중세 판타지 RPG 독서 기록 앱"

### 핵심 컨셉
- 독서 기록 + 게이미피케이션(RPG 요소)을 결합
- 페이지를 읽으면 경험치(EXP)와 골드(Gold)를 획득
- 캐릭터 레벨업, 장비 구매, 칭호(직업) 해금
- 다양한 퀘스트와 업적으로 독서 동기 부여

### 대상 사용자
- 일반인 대상 공개 앱
- 독서 습관을 만들고 싶은 사람
- 게임적 요소로 동기부여가 되는 사람

### 수익화
- 추후 수익화 예정 (구체적 모델 미정, 프리미엄 기능 / 광고 등 검토 중)

## 3. 화면 구성 (5개 탭)

하단 탭 네비게이션으로 5개 화면 전환

### 3-1. 📖 서재 (Library) — 메인 탭

**구성 요소:**
- 퀘스트 패널 (일일/주간/월간 탭 전환)
- 현재 읽는 책 카드 + 페이지 기록 버튼
- 책 목록 (필터: 전체 / 읽는 중 / 완독 / 읽을 책)
- 새 책 추가 폼

**기능 상세:**
- 책 등록: 제목, 저자, 전체 페이지 수, 장르 선택
- 페이지 기록: "여기까지 읽었어요" 방식 (현재 페이지 번호 입력 → 자동 계산)
  - 예: 기존 30p 읽은 상태 → 80p 입력 → "50페이지 읽었습니다" 기록
  - 전체 페이지 도달 시 자동 완독 처리
- 완독한 책은 목록 맨 아래로 정렬 + 회색(desaturated) 처리
- 장르 태그에 "스탯명(카테고리)" 형태로 표시

### 3-2. ⚔️ 캐릭터 (Character)

**구성 요소:**
- 도트 픽셀 캐릭터 (장비에 따라 색상 변화)
- 선택한 칭호 표시 (클릭하면 칭호 선택 모달)
- 레벨 + EXP 바
- 골드 / 총 스탯 포인트 / 획득 칭호 수
- 스탯 레이더 차트 (4각형) + 스탯 바
- 장착 장비 슬롯 (6부위)
- 획득한 칭호 목록 (태그 형태, 클릭으로 선택 가능)
- 다음 칭호 힌트 (잠긴 칭호 중 4개, 진행도 표시)
- 성장 가이드 카드

### 3-3. 🏪 상점 (Shop)

**구성 요소:**
- 보유 골드 표시
- 장비 부위 선택 탭 (6개, 현재 장착 등급 표시)
- 등급별 장비 목록 (Iron ~ Challenger, 7등급)
- 구매 버튼 (골드 차감 → 즉시 장착 → 캐릭터 외견 변화)
- 각 등급에 예상 소요 기간 표시

### 3-4. 🏆 업적 (Achievements)

**구성 요소:**
- 달성/전체 카운트
- 업적 카드 목록 (달성: 녹색 배경, 미달성: 흐림 + 진행도 바)
- 각 업적에 보상 표시

### 3-5. 📊 통계 (Stats)

**구성 요소:**
- 요약 카드 4개 (총 페이지, 완독 수, 연속 독서 일수, 보유 골드)
- 주간 독서량 바 차트 (요일별)
- 장르 분포 (도넛 차트 + 바)
- 월별 요약 (월 클릭 → 해당 월 완독 도서 목록 표시)
- 독서 스트릭 캘린더 (GitHub 잔디 스타일)

### 공통
- 상단 헤더: 앱 로고 + 골드 + 레벨 + 다크모드 토글
- 하단 탭 바: 5개 탭
- 다크모드 지원 (전체 앱)

## 4. 핵심 시스템 설계

### 4-1. 경험치(EXP) & 레벨 시스템

**EXP 획득:**

| 행동 | EXP |
|------|-----|
| 1페이지 읽기 | +1 EXP |
| 책 1권 완독 | +50 EXP (보너스) |
| 독서 메모 작성 | +5 EXP |

**레벨 커브:**
- 공식: 레벨 N → 다음 레벨까지 필요 EXP = round(30 × (N+1)^1.4)
- 누적 EXP 기준 도달 기간 (하루 40p 기준):
  - Lv.5: ~1주
  - Lv.10: ~1개월
  - Lv.15: ~2.5개월
  - Lv.20: ~5개월
  - Lv.25: ~9개월
  - Lv.30: ~14개월

### 4-2. 골드(Gold) 시스템

**골드 획득:**

| 행동 | Gold |
|------|------|
| 1페이지 읽기 | +1 Gold |
| 책 1권 완독 | +30 Gold (보너스) |
| 일일 퀘스트 | +10~30 Gold |
| 주간 퀘스트 | +60~120 Gold |
| 월간 퀘스트 | +250~600 Gold |
| 7일 연속 독서 | +100 Gold |

**일 평균 골드 수입: ~60G** (하루 40p 읽기 + 퀘스트 기준)

### 4-3. 스탯 시스템 (4종)

| 스탯 | 아이콘 | 연결 장르 | 색상 |
|------|--------|----------|------|
| 지혜 (WIS) | 📘 | 비문학 (과학/철학/역사/인문) | #5B8DEF |
| 공감 (EMP) | 📗 | 문학 (소설/에세이/시/전기) | #66BB6A |
| 통찰 (INS) | 📙 | 자기계발 (심리/경영/경제/처세) | #FFA726 |
| 창조 (CRE) | 📕 | 기타 (예술/판타지/SF/만화) | #EF5350 |

**스탯 성장:** 해당 장르 50페이지 읽기 = 해당 스탯 +1

### 4-4. 장비 시스템

**장비 부위 (6종):** 투구, 갑옷, 망토, 무기, 방패, 신발

**장비 등급 (7단계):**

| 등급 | 색상 | 가격 | 예상 소요 기간 |
|------|------|------|-------------|
| Iron | #9E9E9E (회색) | 100G | ~3일 |
| Bronze | #BC8F5A (갈색) | 350G | ~1주 |
| Silver | #90A4AE (은색) | 800G | ~2주 |
| Gold | #FFB300 (금색) | 2,000G | ~1개월 |
| Platinum | #29B6F6 (청색) | 5,000G | ~2.5개월 |
| Master | #AB47BC (보라) | 12,000G | ~6개월 |
| Challenger | #EF5350 (적색) | 30,000G | ~1년+ |

**장비 구매 규칙:**
- 순차적으로만 구매 가능 (Iron → Bronze → Silver → ...)
- 구매 즉시 장착
- 장착 시 도트 픽셀 캐릭터의 해당 부위 색상이 등급 색상으로 변경

### 4-5. 칭호(직업/타이틀) 시스템

직업이 아닌 "칭호"로 운영. 조건 달성 시 자동 해금되며, 사용자가 표시할 칭호를 선택 가능.

**Tier 0 — 시작 (자동 획득)**
- 🌱 견습 모험가: 시작 시 자동

**Tier 1 — 초급 (1~2주)**
- 📖 책벌레: Lv.5 달성
- 🔍 호기심 탐험가: 2가지 장르 읽기

**Tier 2 — 1차 전직 (스탯 기반, ~1개월)**
- 🧙 현자: Lv.10 + 지혜 1위 (15이상)
- 🎵 음유시인: Lv.10 + 공감 1위 (15이상)
- ⚔️ 전략기사: Lv.10 + 통찰 1위 (15이상)
- 🔮 마법사: Lv.10 + 창조 1위 (15이상)

**Tier 2 — 특수**
- 💎 강철 의지: 30일 연속 독서
- ⚡ 속독의 달인: 한 달에 1,500p 이상

**Tier 3 — 2차 전직 순수형 (Lv.20+, ~6-9개월)**
- 📖 대현자: Lv.20 + 지혜 40 이상
- 🎭 전설의 이야기꾼: Lv.20 + 공감 40 이상
- 🛡️ 성기사: Lv.20 + 통찰 40 이상
- ⚡ 대마법사: Lv.20 + 창조 40 이상

**Tier 3 — 2차 전직 하이브리드형 (Lv.20+, 두 스탯 30 이상)**
- 📜 룬 해독사: 지혜 + 공감 ≥ 30
- 🏛️ 왕실 고문관: 지혜 + 통찰 ≥ 30
- 🌌 차원술사: 지혜 + 창조 ≥ 30
- 💫 영혼치유사: 공감 + 통찰 ≥ 30
- 🐉 환수소환사: 공감 + 창조 ≥ 30
- ⚗️ 마법공학사: 통찰 + 창조 ≥ 30

**Tier 3 — 특수**
- 👑 완독의 제왕: 20권 이상 완독
- 🌈 만물박사: 모든 스탯 20 이상

> 참고: 3차 전직 (Lv.40+)은 향후 확장 예정

### 4-6. 퀘스트 시스템

일일 / 주간 / 월간 3개 탭으로 구분.
각 탭마다 3개 퀘스트가 랜덤 생성.
날짜 기반 시드로 같은 날은 같은 퀘스트.

**퀘스트 유형 풀 (25종+):**

**일일 (갱신: 매일 자정)**
- 페이지 기반: "오늘 N페이지 읽기", "평균+10% 도전"
- 특정 책: "OO책 Xp까지 읽기"
- 기록: "독서 기록 남기기", "독서 메모 작성", "앱 접속하기"
- 장르: "특정 장르 책 읽기"
- 보상: 10~30 Gold

**주간 (갱신: 매주 월요일)**
- 총 페이지: "이번 주 Np 읽기 (평균+10%)"
- 완독: "이번 주 1권 완독"
- 연속: "7일 연속 독서", "5일 이상 독서"
- 장르: "2가지 이상 장르 읽기"
- 특정 책: "OO책 Np 더 읽기"
- 메모: "독서 메모 3개 작성"
- 보상: 60~120 Gold

**월간 (갱신: 매월 1일)**
- 총 페이지: "이번 달 Np 읽기 (평균+10%)"
- 완독: "이번 달 N권 완독"
- 연속: "20일 이상 독서", "15일 연속"
- 장르: "3가지 이상 장르", "새로운 장르 도전"
- 누적: "누적 N p 달성"
- 특정 책: "OO책 완독하기"
- 보상: 250~600 Gold

**핵심:** 사용자의 최근 평균 독서량을 기반으로 10% 증가된 도전 퀘스트를 자동 생성

## 5. 데이터베이스 설계 (Supabase)

### 테이블 구조

```sql
-- 사용자 프로필
users
  id (uuid, PK, auth.users 연결)
  nickname (text)
```
